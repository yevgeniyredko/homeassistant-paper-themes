# Paper Theme for Home Assistant

When I set up the [Frosted Glass theme](https://github.com/wessamlauf/homeassistant-frosted-glass-themes), I found that I like its look more without card-mod styles. This is a fork of the theme with card-mod styles removed.

## Installation

1. Install [HACS](https://hacs.xyz/)
2. Add https://github.com/yevgeniyredko/homeassistant-paper-themes/ to custom repositories, and select `Theme` as the category
3. Find `Paper` and install it
4. Go to your personal account settings and select Paper as the theme
